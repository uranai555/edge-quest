# 実装順序 — 画面端クエスト v0.1 MVP

## 全体方針

**右端が完了条件のMilestone。左から順に進める。**

```
Milestone 0 ──→ Milestone 1 ──→ Milestone 2 ──→ Milestone 3 ──→ Milestone 4
(企画固定)     (動画検証素材)   (技術コア検証)   (MVP APK完成)    (テスト配布)
```

**パラレルで進められるものは同時進行する。**
Hermes（私）が全体設計・調整を担当し、Codexに実装、Devinに調査を並行依頼する。
