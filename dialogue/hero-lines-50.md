# ユウ（勇者）初期台詞50個

> コード上は `HeroYu`、全台詞は `object HeroLines` または JSON で管理。
> 各台詞は `Reaction` エンティティとして保持。タップ台詞はcooldown=5s、時間帯系はcooldown=15m。

---

## 台詞追加ルール（コーディング用）

```kotlin
data class HeroLine(
    val id: Int,
    val text: String,
    val category: HeroLineCategory,
    val minEvolutionStage: Int = 1,   // この台詞を出すのに必要な最低進化段階
    val cooldownSeconds: Long,         // 同じカテゴリの再表示間隔
    val isOneShot: Boolean = false     // trueの場合、一度表示したら二度と表示しない
)

enum class HeroLineCategory {
    TAP, MORNING, AFTERNOON, EVENING, NIGHT,
    LOW_BATTERY, IDLE_RETURN, LONG_USAGE,
    LEVEL_UP, RANDOM
}
```
